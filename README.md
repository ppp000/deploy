若是学生身份看这里：200大洋前后，一年的ecs服务器使用期，细节自行了解阿里云云翼计划。
不是学生身份看这里：注册后阿里云会赠送一个月的ecs服务器，再购买两个月即可（必须购买，若不购买则无备案服务号，则无法备案）
